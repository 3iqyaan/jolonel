# Jolonel
A task manager that simplifies getting things done.
## How does it help?
I believe that all ambitious goals can be made tangible and achievable by breaking them down into smaller tasks. Jolonel is designed to facilitate this process by allowing users to create tasks, assign them to specific goals, and organize them using tags among other features.
## How does it work?
There are three modes of operation:
1. **Do** - To Create, Start, Edit, Complete, and Delete tasks.
2. **View** - To View tasks Filtered, Sorted, Graphed, and thereby the progress Analyzed and learned from.
3. **Change** - To *Create, Edit, and Delete* Tags, Goals, Schedules, Recurring Tasks, Estimated durations, and Priority levels for tags and goals.
